REST API Documentation

All endpoints require HTTP Basic Authentication

Authentication

All /api/* endpoints require HTTP Basic Auth:

Authorization: Basic <base64(username:password)>

Default credentials:

Username: admin
Password: admin123

Set API_USERNAME and API_PASSWORD environment variables in production.

Base URL & Response Codes
http://localhost:5000
200Validation passed
201Template created
422Validation failed (with details)
401Authentication required
404Template not found
409Template name conflict
GET /api/templates List all templates
curl -u admin:admin123 http://localhost:5000/api/templates
POST /api/templates Create a template
curl -u admin:admin123 -X POST http://localhost:5000/api/templates \ -H "Content-Type: application/json" \ -d '{ "name": "My Template", "description": "Optional description", "rules": [ { "field_path": "customer.email", "display_name": "Customer Email", "validations": [ {"type": "required"}, {"type": "email"} ] }, { "field_path": "order.amount", "validations": [ {"type": "required"}, {"type": "odata_edm_decimal", "precision": 13, "scale": 2}, {"type": "min_value", "value": 0} ] } ], "group_rules": [ { "name": "Phone or Email required", "fields": ["customer.phone", "customer.email"], "rule_type": "at_least_one" } ] }'
POST /api/validate/{id} Validate JSON against template by ID
curl -u admin:admin123 -X POST http://localhost:5000/api/validate/1 \ -H "Content-Type: application/json" \ -d '{"customer": {"email": "john@example.com"}, "order": {"amount": 99.99}}'

Example response (200 — valid):

{ "valid": true, "template_name": "My Template", "summary": {"total_fields": 2, "passed": 2, "failed": 0, ...}, "field_results": [...], "group_results": [...] }

Example response (422 — invalid):

{ "valid": false, "template_name": "My Template", "summary": {"total_fields": 2, "passed": 1, "failed": 1, ...}, "field_results": [ {"field_path": "customer.email", "valid": false, "errors": ["Invalid email address: bad-email"], ...} ], ... }
POST /api/validate/by-name/{name} Validate JSON against template by name
curl -u admin:admin123 -X POST "http://localhost:5000/api/validate/by-name/My%20Template" \ -H "Content-Type: application/json" \ -d '{"customer": {"email": "john@example.com"}}'
PUT /api/templates/{id}   DELETE /api/templates/{id}
# Update curl -u admin:admin123 -X PUT http://localhost:5000/api/templates/1 \ -H "Content-Type: application/json" -d '{...same as POST...}' # Delete curl -u admin:admin123 -X DELETE http://localhost:5000/api/templates/1
